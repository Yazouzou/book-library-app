# Book Library Application

## Overview
The Book Library Application is a modern React-based web app that allows users to search and explore books using the Open Library API. Users can search by title or author, view book details, and browse results in a clean, responsive interface.

This project focuses on building a responsive front-end application using modern web technologies.

## Features Implemented
- Search books by title or author
- Display book cover, title, author, publisher
- Book rating, book reviews
- View detailed book information
- Responsive design using Tailwind CSS
- Error handling for empty results and network issues

## Technologies Used
- React (Vite)
- Tailwind CSS
- Open Library API
- JavaScript (ES6)
- React Router

## Installation
1. Clone the repository : https://github.com/Yazouzou/book-library-app.git
2. Navigate to the project folder
3. cd book-library-app
4. Install dependencies
5. npm install
6. Start development server
7. npm run dev
8. Open the application in your browser:
http://localhost:5173

## API Used

This project uses the Open Library API to fetch book data.

## API Documentation:
https://openlibrary.org/developers/api

Example request:

https://openlibrary.org/search.json?q=harry+potter
