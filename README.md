# Book Library Application

## Overview
This is a React-based Book Library application that allows users to search for books using the Open Library API.

## Features Implemented
- Search books by title or author
- Display book cover, title, author, publisher
- View detailed book information
- Responsive design using Tailwind CSS
- Error handling for empty results and network issues

## Technologies Used
- React (Vite)
- Tailwind CSS
- Open Library API
- JavaScript (ES6)

## Installation
1. Clone the repository
2. Run npm install
3. Run npm run dev
